# DCE_Manager.exe

DCE_Manager.exe is a utility designed to simplify the management of DCE (Dynamic Campaign Engine) campaigns for DCS World.

It provides a graphical interface to perform many common tasks that would otherwise require manually editing files or navigating through the DCE folder structure.

---

## Features

Current features include:

* Install and manage DCE campaigns
* Clone existing campaigns
* Update DCE_Manager
* Update DCE ScriptMods
* Update supported campaigns
* Edit campaign settings

New features are added regularly as DCE continues to evolve.

---

## Installation

If you download a complete DCE campaign from CEF, everything you need is already included:

* The campaign
* DCE ScriptMods
* DCE_Manager.exe

For most users, no additional download is required.

This repository is mainly intended for:

* DCE_Manager updates
* Development
* Source code

🔒 Privacy & Analytics

DCE Manager includes basic, optional usage analytics to help improve the application.

What is sent (at most once a day):

    A unique client ID (generated via SHA-256 hashing, with no personal or nominative data).

    The total number of times the app has been launched.

    The current versions of the application and scripts.

Your Control:
This data collection is entirely optional. You can disable it at any time by unchecking the dedicated box in the application's options. No data is ever shared with third parties.
